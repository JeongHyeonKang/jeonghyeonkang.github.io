```
02-11 Pods with YAML
```

```
[yaml 기본 구조]
pod-definition.yml

apiVersion: 
kind:
metadata:
    name:
    lables:
        app:
        type:
spec:
    containers:
        - name:
          image:
```
```
[Pod 생성]
$ kubectl create -f pod-definition.yml
$ kubectl apply -f pod.yaml
```
