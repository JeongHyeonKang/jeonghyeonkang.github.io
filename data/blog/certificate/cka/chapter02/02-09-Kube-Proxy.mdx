```
02-09 Kube-Proxy
```

```
Kubernetes 클러스터 내에서 모든 포드는 다른 모든 포드에 연결할 수 있으며 이는 포드 네트워킹 클러스터를 클러스터에 배포하여 수행됩니다.
Kube-Proxy는 Kubernetes 클러스터의 각 노드에서 실행되는 프로세스입니다.
```
