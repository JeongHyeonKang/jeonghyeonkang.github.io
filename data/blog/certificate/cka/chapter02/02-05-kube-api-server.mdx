```
02-05 kube-api-server
```

```
[kube-apiserver 설치]
$ wget https://storage.googleapis.com/kubernetes-release/release/v1.13.0/bin/linux/amd64/kube-apiserver
```
```
[kube-apiserver 보기 - Kubeadm]
kubeadm은 kube-apiserver를 마스터 노드의 kube-system 네임스페이스에 포드로 배포합니다.
$ kubectl get pods -n kube-system
```
``` 
[kube-apiserver 옵션 보기 - Kubeadm]
다음 위치에 있는 포드 정의 파일에서 옵션을 볼 수 있습니다./etc/kubernetes/manifests/kube-apiserver.yaml
$ cat /etc/kubernetes/manifests/kube-apiserver.yaml
```
```
[kube-apiserver 옵션 보기 - 수동]
Kubeadm이 아닌 설정에서는 kube-apiserver.service를 확인하여 옵션을 검사할 수 있습니다.
$ cat /etc/systemd/system/kube-apiserver.service
```
